# jsse_generateSuperellipsePath

Генерирует SVG‑путь (атрибут `d`) для суперэллипса с регулируемой формой углов.  
Форма меняется непрерывно от вогнутых прямоугольных углов до выпуклых прямоугольных углов, проходя через скос, круглые углы и классический суперэллипс.  
Аппроксимация выполняется кубическими кривыми Безье с плавным переходом между всеми промежуточными состояниями.

## Синтаксис

```js
import { jsse_generateSuperellipsePath } from 'f4n70m/sj-superellipse';

const pathData = jsse_generateSuperellipsePath(width, height, radius, curveFactor, precision);
```

## Параметры

| Имя           | Тип     | По умолчанию | Описание |
|---------------|---------|--------------|----------|
| `width`       | number  | –            | Ширина фигуры (в пикселях). |
| `height`      | number  | –            | Высота фигуры (в пикселях). |
| `radius`      | number  | –            | Радиус скругления углов. Автоматически ограничивается половиной меньшей стороны. |
| `curveFactor` | number  | –            | Коэффициент формы углов. Диапазон: **от -2 до 2**. |
| `precision`   | number  | `2`          | Количество знаков после запятой в координатах SVG. |

## Параметр `curveFactor`

Значения `curveFactor` определяют форму каждого угла:

| Значение      | Форма угла                              |
|---------------|-----------------------------------------|
| `-2`          | Вогнутые прямоугольные углы              |
| `-G` (≈ -0.5523) | Вогнутые круглые углы                 |
| `0`           | Прямой скос (без скругления)             |
| `G` (≈ 0.5523)  | Выпуклые круглые углы (четверть окружности) |
| `1`           | Выгнутые суперэллипсные углы (классический суперэллипс) |
| `2`           | Выгнутые прямоугольные углы               |

> **Примечание:**  
> `G = (4/3) * (√2 - 1) ≈ 0.5522847498` — константа, при которой кубическая кривая Безье аппроксимирует четверть окружности.  
> Для промежуточных значений `curveFactor` форма плавно интерполируется между указанными эталонами.

## Возвращаемое значение

Строка (`string`) с SVG‑командами для элемента `<path>`.  
Пример: `"M 0,10 L 0,5 C 0,2 3,0 5,0 L 15,0 L 20,0 C 23,0 25,2 25,5 L 25,15 Z"`

## Примеры

## Прямоугольник со скруглёнными углами (curveFactor = G)

```js
const path = jsse_generateSuperellipsePath(200, 150, 30, 0.5523);
document.querySelector('path').setAttribute('d', path);
```

## Суперэллипс с curveFactor = 1

```js
const path = jsse_generateSuperellipsePath(200, 150, 30, 1);
```

## Вогнутый квадрат с curveFactor = -2

```js
const path = jsse_generateSuperellipsePath(200, 200, 40, -2);
```

## Изменение точности координат

```js
const path = jsse_generateSuperellipsePath(100, 80, 15, 0.8, 3); // 3 знака после запятой
```

## Особенности реализации

- Радиус автоматически ограничивается, чтобы углы не наезжали друг на друга.
- Для разных пропорций фигуры (`width` ≠ `height`) радиусы по горизонтали и вертикали могут немного отличаться, сохраняя визуально согласованное скругление.
- Все вычисления используют кубические кривые Безье, что обеспечивает высокое качество аппроксимации при малом количестве команд в пути.
- Функция экспортируется как `jsse_generateSuperellipsePath` и может быть импортирована из пакета.

## Связанные функции

- `jsse_roundf(value, precision)` – округление числа с заданной точностью.
- `jsse_getBorderRadiusFactor()` – возвращает константу `G = (4/3)*(√2-1)`.
