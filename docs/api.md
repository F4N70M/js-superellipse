## API: инициализация и управление суперэллипсами

Файл `api.js` предоставляет удобный интерфейс для подключения и управления суперэллипсами на элементах DOM. Все экспортируемые значения и методы задокументированы ниже.

### Экспортируемые объекты

#### `jsse_controllers`
`WeakMap`, который хранит связь между DOM-элементом и его экземпляром контроллера (`SuperellipseController`).  
**Обычно не требуется использовать напрямую** — он используется внутри библиотеки.

---

### Расширения прототипа `Element`

#### `element.superellipse` (геттер)
Возвращает экземпляр контроллера, связанный с элементом, или `undefined`, если элемент не инициализирован.

**Пример:**
```js
const div = document.querySelector('.my-div');
div.superellipseInit({ r: 0.5 });
console.log(div.superellipse); // экземпляр SuperellipseController
```

#### `element.superellipseInit(options)`
Инициализирует суперэллипс на элементе. Если элемент уже был инициализирован, предыдущий контроллер уничтожается, и создаётся новый.

**Параметры:**
- `options` — объект настроек, передаваемый в конструктор `SuperellipseController`. Подробнее см. в документации контроллера.

**Возвращает:** сам элемент (`this`) для цепочечных вызовов.

**Пример:**
```js
document.querySelector('.box').superellipseInit({
  r: 0.7,
  stroke: '#ff0000',
  fill: 'transparent'
});
```

---

### Глобальная функция

#### `superellipseInit(target, options)`
Универсальная функция для инициализации суперэллипсов на одном или нескольких элементах.

**Параметры:**
- `target` — может быть:
  - **CSS-селектором** (строка) — инициализирует все найденные элементы;
  - **DOM-элементом** (`Element`) — инициализирует один элемент;
  - **коллекцией элементов** (`NodeList`, `Array` из `Element`) — инициализирует каждый элемент коллекции.
- `options` — объект настроек для контроллера.

**Возвращает:**
- для селектора — массив контроллеров для всех найденных элементов;
- для одного элемента — экземпляр контроллера этого элемента;
- для коллекции — массив контроллеров для всех элементов коллекции.

**Примеры:**
```js
// Инициализация по селектору
const controllers = superellipseInit('.card', { r: 0.5 });

// Инициализация одного элемента
const elem = document.getElementById('hero');
const controller = superellipseInit(elem, { r: 0.8 });

// Инициализация коллекции
const items = document.querySelectorAll('.item');
const itemControllers = superellipseInit(items, { stroke: 'blue' });
```

---

### Примечания
- При повторном вызове `superellipseInit` на уже инициализированном элементе старый контроллер автоматически уничтожается (вызывается его метод `destroy`).
- Функция `superellipseInit` выбрасывает ошибку, если `target` имеет неподдерживаемый тип.
- Для доступа к контроллеру после инициализации используйте геттер `element.superellipse`.
