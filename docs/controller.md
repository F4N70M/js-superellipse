# Класс `SuperellipseController`

Класс `SuperellipseController` отвечает за применение скруглённой формы суперэллипса к любому DOM-элементу. Он автоматически отслеживает изменения размеров, стилей и видимости элемента, пересчитывает и применяет соответствующие CSS-свойства или SVG-слой.

---

## 🔧 Конструктор

```javascript
new SuperellipseController(element, options)
```

**Параметры**

| Имя       | Тип     | Описание                                                                 |
|-----------|--------|--------------------------------------------------------------------------|
| `element` | `Element` | DOM-элемент, к которому применяется суперэллипс.                         |
| `options` | `Object` | (необязательно) Настройки контроллера.                                   |

**Опции**

| Свойство       | Тип       | По умолчанию       | Описание                                                                                           |
|----------------|----------|--------------------|----------------------------------------------------------------------------------------------------|
| `mode`         | `string` | `'svg-layer'`      | Режим реализации: `'svg-layer'` (добавляет слой SVG) или `'clip-path'` (использует CSS `clip-path`). |
| `curveFactor`  | `number` | из `jsse_getBorderRadiusFactor()` | Коэффициент кривизны суперэллипса. Чем выше значение, тем ближе к прямоугольнику.                  |
| `precision`    | `number` | `2`                | Количество знаков после запятой в сгенерированном пути.                                            |
| `autoResize`   | `boolean`| `true`             | Автоматически отслеживать изменения размеров элемента через `ResizeObserver`.                      |

---

## 🧩 Публичные методы

## `switchMode(mode)`

Переключает режим работы контроллера.

- **Аргументы:**  
  `mode` (`string`) — новый режим (`'svg-layer'` или `'clip-path'`).

- **Возвращает:**  
  `boolean` — `true`, если переключение выполнено успешно, иначе `false`.

- **Пример:**
  ```javascript
  controller.switchMode('clip-path');
  ```

## `destroy()`

Полностью уничтожает контроллер: удаляет все обёртки, сбрасывает стили, отключает наблюдателей и удаляет ссылку из глобального реестра.

- **Пример:**
  ```javascript
  controller.destroy();
  ```

## `setCurveFactor(value)`

Устанавливает новый коэффициент кривизны и обновляет форму.

- **Аргументы:**  
  `value` (`number`) — новое значение кривизны.

- **Пример:**
  ```javascript
  controller.setCurveFactor(2.5);
  ```

## `setPrecision(value)`

Устанавливает точность генерации пути (количество знаков после запятой).

- **Аргументы:**  
  `value` (`number`) — новое значение точности.

- **Пример:**
  ```javascript
  controller.setPrecision(4);
  ```

## 🎨 Режимы работы

## `'svg-layer'` (по умолчанию)

- Создаёт внутри элемента дополнительный слой `<svg>` с абсолютным позиционированием.
- Все фоновые стили (`background`, `border`, `box-shadow`) применяются к этому SVG-слою, а не к исходному элементу.
- Оригинальное содержимое оборачивается в `<div>` с относительным позиционированием, чтобы не перекрывать SVG-слой.

## `'clip-path'`

- Использует CSS-свойство `clip-path: path('...')` для непосредственного обрезания элемента.
- Все стили элемента сохраняются (применяются к самому элементу).
- Не требует добавления дополнительных DOM-узлов.

---

## 🧪 Управляемые CSS-свойства

Контроллер отслеживает и управляет следующими стилями:

| Свойство          |
|-------------------|
| `border-radius`   |
| `background`      |
| `background-color`|
| `background-image`|
| `border`          |
| `box-shadow`      |
| `position`        |

При смене режима контроллер корректно переносит значения этих свойств между элементом и SVG-слоем.

## 🔍 Внутренние механизмы

- **Слежение за стилями (`WeakMap`)**  
  Исходные и текущие значения управляемых свойств сохраняются в `WeakMap`, что позволяет восстановить их после отключения режима.

- **Наблюдатели (`Observers`)**  
  - `MutationObserver` — реагирует на изменения атрибутов `style` и `class`.
  - `ResizeObserver` — отслеживает изменение размеров элемента (если `autoResize: true`).
  - `IntersectionObserver` — обновляет форму, если элемент стал видимым после скрытия (`display: none`).
  - `MutationObserver` на `document.body` — автоматически вызывает `destroy()` при удалении элемента из DOM.

- **Пересчёт пути**  
  Путь суперэллипса генерируется функцией [jsse_generateSuperellipsePath](./core.md) с учётом текущих размеров, радиуса скругления (`border-radius`), коэффициента кривизны и точности.

## 📦 Пример использования

```javascript
import { SuperellipseController } from './controller.js';

const element = document.querySelector('.my-element');
const controller = new SuperellipseController(element, {
  mode: 'svg-layer',
  curveFactor: 1.5,
  precision: 3
});

// Позже можно изменить кривизну
controller.setCurveFactor(2.0);

// Переключить режим
controller.switchMode('clip-path');

// При необходимости уничтожить
controller.destroy();
```

---

## ⚠️ Примечания

- Класс автоматически удаляет себя, когда элемент удаляется из DOM.
- При временном скрытии элемента (`display: none`) обновления приостанавливаются и возобновляются после появления.
- Для корректной работы в режиме `svg-layer` важно, чтобы элемент имел явную ширину и высоту (например, через CSS или атрибуты).
